# Конвертер учебного конфигурационного языка в TOML
**Данильченко Д.А ИКБО-68-24** | **Вариант 10**

## 1. Цель работы

Реализовать инструмент командной строки на Python, который:

- читает файл на **учебном конфигурационном языке**;
- выполняет **лексический и синтаксический разбор**;
- вычисляет **константы и константные выражения** на этапе трансляции;
- преобразует результат в формат **TOML** и записывает его в выходной файл;
- при синтаксических и семантических ошибках выводит сообщения.

---

## 2. Запуск программы

- параметр ``-i`` входной файл
- параметр ``-o`` выходной файл Toml

```bash
python app.py -i example.conf -o output.toml
```
